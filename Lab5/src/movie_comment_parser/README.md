# Movie Comment Parser

This is a simple example parser package. It extracts:

1. Russian title
2. English title
3. Year

From dataset movie comments.

---

## Dataset structure:

```markdown
dataset/
├── good/
│   ├── 0000.txt
│   ├── 0001.txt
│   │   ...
│   └── 0999.txt
└── bad/
    ├── 0000.txt
    ├── 0001.txt
    │   ...
    └── 0999.txt
```

---
